ejercicio 3: JavaScript y el DOM

¿Qué es el DOM?
El DOM (Document Object Model) es la estructura del documento web como un árbol de objetos que JavaScript puede manipular.

Funciones básicas para manipular el DOM:

document.getElementById()       |→ Selecciona por ID.
document.querySelector()        |→ Selecciona con selector CSS.
element.appendChild()           |→ Agrega nodo hijo.
element.remove()                |→ Elimina nodo.
element.style.propiedad         |→ Cambia estilos.


Ejemplo práctico:

  id="demo">Texto original
  onclick="document.getElementById('demo').innerText = 'Texto cambiado';">
    Cambiar texto con JavaScript
